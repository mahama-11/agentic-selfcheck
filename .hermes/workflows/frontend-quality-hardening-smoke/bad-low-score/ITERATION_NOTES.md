## Review feedback for v1

Reviewer/visual QA findings:

1. Layout foundation is good, but selected workstream is not strongly connected to center/right panels.
2. Central board hierarchy is clear but still static and card-heavy.
3. Right evidence inspector should better answer: what should I do next?
4. Evidence chips look clickable but do not communicate behavior.
5. Status rows should feel more operational, with drill-down/detail affordance.

## Iteration goals for v2

- Add explicit selected workstream context.
- Add recommended next action panel.
- Add stage/step rhythm instead of flat rows.
- Make evidence files and action semantics clearer.
- Add density toggle and stronger interaction affordance.
